# Welcome to AccessBank

Learn Flutter accessibility by building a real banking app.

## What You'll Build

AccessBank is a Flutter banking app that you'll make fully accessible step by step. Each chapter focuses on a specific accessibility topic, walking you through the "why" and the "how" with real code changes.

## What You'll Learn

- Setting up accessibility tooling in Flutter
- Writing semantic widget trees with `Semantics`
- Navigation and focus management for screen readers
- Visual accessibility: contrast, text scaling, and touch targets
- Accessible forms and input validation
- Reducing motion for vestibular disorders
- Live regions and announcements
- Testing with TalkBack, VoiceOver, and Flutter's accessibility tools
- Polishing for production

## How to Use This Tutorial

Work through the chapters in order. Each chapter has:

- **Why this matters** — the accessibility principle behind the topic
- **Step-by-step code changes** — applied directly to the AccessBank app
- **Try it yourself** — checkpoints to verify with a real screen reader
- **What you fixed** — a summary of what changed

Ready? Start with [Chapter 0: Setting Up Your Toolkit](/chapters/toolkit).
